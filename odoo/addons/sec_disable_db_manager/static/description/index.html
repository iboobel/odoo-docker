Disable Database Manager

Backport of same feature from Odoo 11.

This is a backport from Odoo 11 which disables the Odoo database manager page and database management functions via XML/JSON RPC if Odoo is started with the --no-database-list CLI parameter or the list_db configuration parameter set to False.

Plug & Play

No configuration required — just install the module and forget it!

Need to re-enable database manager?

It's easy — simply run Odoo without the --no-database-list CLI parameter and/or set the db_list config value to True.
